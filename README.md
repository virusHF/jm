## 周公解梦境 for 微信小程序


## 简介
使用的微信云开发，不需要自己搭建服务器

1.根据梦境搜索结果
2.根据分类查看梦境
3.收藏梦境
4.分享梦境

## 使用方式
1.替换自己的appkey
2.极速api注册账号，替换请求的key
